# 장마 전선 실험실 — 예선 제출 기획서

**2026 기상·기후 AI 해커톤 경진대회 / 개인 출품**

> 한 줄 소개 — 한반도 지도 위에서 장마 저기압을 직접 움직이며, AI 서러게이트 모델이
> 실시간으로 강수를 추론하고 그 결과를 과학적으로 해석해 주는 **체험형 기상·기후 학습 웹앱**.

- **체험 링크(데모)**: GitHub Pages 정적 데모 — https://ahagoitda.github.io/weather/
- **형태**: 설치·로그인·서버 없이 브라우저에서 즉시 동작하는 웹앱

---

## 1. 문제 정의 (주제 적합성)

- 2020~2025년 장마 집중호우로 인한 피해가 매년 반복되고, 기후변화로 극한 호우의 빈도·강도가 커지고 있습니다.
- 그러나 학교 현장의 기상·기후 교육은 대부분 **정적 지도와 영상 설명**에 그쳐, 학생이 장마 전선의
  이동·강수 생성 원리를 **직접 실험**할 기회가 없습니다.
- 본 작품은 기상(장마 전선)과 기후(해수면온도 상승)를 **하나의 조작 가능한 모델**로 묶어,
  학생이 "왜 어떤 해에는 장마가 더 위험한가"를 스스로 발견하도록 설계했습니다.

## 2. 솔루션 개요 (체험·참여형 설계)

한반도 지도에서 **저기압 마커를 드래그**하고 **3개 슬라이더**(전선 강도·이동 속도·해수면온도)를
조작하면, 모델이 즉시 8개 도시의 강수를 다시 계산하고 지도·차트·AI 해석이 실시간으로 바뀝니다.

**"조작 → 즉시 피드백 → 재조작"의 능동 탐구 루프**가 핵심이며, 다음 두 모드로 참여형을 강화했습니다.

- **재현 챌린지(미션)** — "2023 중부 집중호우를 재현하라" 같은 과제를 주고, 학생이 전선을 조작해
  실제 사례의 강수 패턴에 얼마나 가까운지를 **유사도(%)로 채점**합니다. 방향 힌트를 주되 정답값은
  숨겨 탐구를 유도하고, 성공 시 뱃지를 부여합니다.
- **기후변화 시나리오** — 현재 / +1.5°C / +3.0°C로 미래 해양 상태를 토글하면, **같은 전선이라도**
  장마가 몇 % 강해지는지 즉시 체감합니다.

## 3. 학습 효과

- 모든 조작에 대해 **AI 교육 피드백**(핵심 요약 + 실제 관측 비교 + 학습 포인트)이 즉시 제공됩니다.
- **설명가능한 요인 분해(XAI)** 패널이 강도·수온·정체·위치가 강수에 기여한 비중을 보여줘
  "블랙박스가 아닌" 이해를 돕습니다.
- 다루는 핵심 개념: 저기압·수증기·전선 정체·해수면온도(Clausius–Clapeyron, 1°C↑→수증기 ~7%↑)·
  기후변화와 극한 강수의 관계.

## 4. 기존과의 차별점 (아이디어 독창성)

| 기존 기상 교육 콘텐츠 | 장마 전선 실험실 |
|---|---|
| 정적 지도·영상으로 **보여주기** | 직접 조작하고 **채점받는** 능동 탐구 |
| 정답을 일방 전달 | 가설→실험→유사도 검증의 과학탐구 게임화 |
| '기상' 현상 설명에 한정 | '기후'(미래 온난화) 시나리오까지 연결 |
| 결과만 표시(블랙박스) | 요인 분해(XAI)로 원리까지 설명 |

→ **"게임화된 과학탐구 루프 + 기후 렌즈 + 설명가능성"** 의 결합이 본 작품의 독창적 지점입니다.

## 5. 실현 가능성 (이미 동작하는 프로토타입)

- **즉시 동작**: 정적 웹앱(Next.js static export)으로, 심사 현장에서 **브라우저만 열면** 추론이
  실행됩니다. 서버·설치·API 키·로그인 불필요 → 시연 실패 위험이 거의 없습니다.
- **AI 추론의 경량화**: 무거운 수치예보(NWP) 대신 기상 물리를 반영한 **경량 서러게이트(대리) 모델**을
  브라우저에서 실행해 지연 없이 동작합니다. *(서러게이트 모델은 기상·기후 ML에서 널리 쓰이는 기법입니다.)*
- **데이터 신뢰성**: 사례 비교의 기준값은 임의의 가짜 관측치가 아니라 **같은 모델을 그 사례의 대표
  조건으로 돌린 값**이라 자기일관적이며, 화면에는 **공개 보도·기상청 기반의 정성적 기록과 출처**를 함께
  표기합니다.
- **확장성**: 서버 배포(Vercel) 시 생성형 LLM(Gemini/Grok) 해석을 켤 수 있도록 구조를 분리해
  두었습니다(환경변수 토글).

### AI 적용 구조 (3계층)
1. **서러게이트 모델** — 입력(위치·강도·속도·수온)→강수 출력을 경량 함수로 에뮬레이트.
2. **설명가능 요인 분해(XAI)** — 각 입력의 기여도를 분해해 제시.
3. **생성형 LLM 보강(선택)** — 서버 배포 시 자연어 해석 생성.
